Block 요소 vs Inline 요소

HTML 요소는 기본적으로 어떻게 공간을 차지하느냐에 따라 두 가지 유형으로 나뉩니다. 아래에서 각 유형의 동작 방식과 대표 태그를 살펴보세요.

Block 요소 — 새 줄에서 시작, 가로폭 100%
Inline 요소 — 콘텐츠 흐름 안에서 나란히
block <div>

범용 컨테이너로, 의미 없이 여러 요소를 묶을 때 사용합니다. 항상 새 줄에서 시작하며 부모 너비를 꽉 채웁니다.

첫 번째 div — 한 줄을 모두 차지합니다
두 번째 div — 자동으로 아래로 내려갑니다
세 번째 div — 각각 독립된 행에 위치합니다
<div>첫 번째 div</div> <div>두 번째 div</div> /* 각 div는 새 줄에서 시작 */
inline <span>

범용 인라인 컨테이너로, 텍스트 일부에 스타일을 적용할 때 사용합니다. 줄을 바꾸지 않고 텍스트 흐름 안에 배치됩니다.

이 문장에서 span 영역은 텍스트 흐름 안에 그대로 녹아들어 있습니다. 두 번째 span도 마찬가지입니다.
텍스트 안의 <span>강조 부분</span> 계속 이어집니다. /* 줄 바꿈 없이 인라인 배치 */
block <p> / <h1>~<h6>

단락과 제목 태그입니다. 브라우저가 위아래 여백(margin)을 기본으로 추가하며, 새 줄에서 시작합니다.

<h2> 제목 태그
<p> 단락 텍스트입니다. 블록 요소이므로 위아래로 여백이 생기고 새 줄에서 시작합니다.
<p> 또 다른 단락 — 자동으로 다음 줄로 이동합니다.
<h2>제목</h2> <p>첫 번째 단락</p> <p>두 번째 단락</p>
inline <a> <strong> <em> <code>

링크, 굵은 글씨, 기울임꼴, 인라인 코드 태그입니다. 모두 텍스트 흐름을 유지하며 나란히 배치됩니다.

<a> 링크 다음에 <strong> 강조, 그리고 <em> 기울임과 <code> 코드가 모두 한 줄에 나란히 표시됩니다.
<a>링크</a> <strong>강조</strong> <em>기울임</em> <code>코드</code> /* 모두 인라인 — 줄 바꿈 없음 */
block <ul> / <ol>

목록 태그입니다. <ul>은 순서 없는 목록, <ol>은 순서 있는 목록이며 각각 블록을 형성합니다.

<ul> 순서 없는 목록
  • 항목 A
  • 항목 B
  • 항목 C
<ul> <li>항목 A</li> <li>항목 B</li> </ul>
inline <img> <input> <button>

이미지, 입력창, 버튼도 기본적으로 인라인 요소입니다. 텍스트와 함께 같은 줄에 흐릅니다. (button은 inline-block에 가깝습니다.)

텍스트 옆에 🖼 img [ input ] 버튼 이 모두 나란히 옵니다.
텍스트 <img src="..."> <input type="text"> <button>클릭</button> /* 세 요소 모두 인라인 배치 */
Block vs Inline — 나란히 비교 동일한 콘텐츠, 다른 배치 방식

아래 점선 박스는 각 요소가 실제로 어떻게 공간을 차지하는지 시각화합니다.

블록 요소 — 가로 전체를 차지합니다 블록 요소 — 항상 새 줄에서 시작합니다
인라인 요소는 → 인라인 A 그리고 인라인 B 나란히 흐릅니다.
특성 Block 요소 Inline 요소
새 줄 시작 ✓ 항상 새 줄 ✗ 흐름 유지
기본 너비 부모 너비 100% 콘텐츠 크기만큼
width / height 지정 ✓ 가능 ✗ 무시됨
margin / padding 4방향 모두 적용 좌우만 완전 적용
대표 태그 div, p, h1~h6, ul, ol, section, article span, a, strong, em, img, input, button
CSS display 속성으로 동작 바꾸기

CSS display 속성을 사용하면 원래 유형과 관계없이 블록 / 인라인 동작을 변경할 수 있습니다. inline-block은 인라인처럼 흐르면서도 width/height를 지정할 수 있는 중간 형태입니다.

span { display: block }
span이지만 블록처럼 새 줄을 차지합니다 두 번째 span도 아래로 내려갑니다
div { display: inline }
텍스트와
div지만 인라인
처럼 나란히 배치됩니다.
display: inline-block
나란히 + 크기 지정:
박스 A
140px 너비
박스 B
140px 너비
/* inline-block: 인라인처럼 흐르지만 width/height 지정 가능 */ span { display: block; } /* span → 블록 동작 */ div { display: inline; } /* div → 인라인 동작 */ div { display: inline-block; } /* 중간 형태 */
