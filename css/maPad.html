Margin vs Padding

margin은 요소 바깥의 여백, padding은 요소 안쪽의 여백입니다.

margin — 바깥 여백

margin 영역 요소(box)
box  ·  margin: 30px

노란 배경 = margin 공간 / 파란 박스 = 실제 요소

padding — 안쪽 여백

padding 영역 (배경색 포함) 요소(box)
box  ·  padding: 30px

초록 배경 = padding 공간(요소의 일부) / 파란 박스 = 콘텐츠
